### 1. * 읽는 법

포인터로 읽으려면 자료형이 있어야한다.
```cpp
int* pA // pA는 int type의 주소를 저장하는 포인터다

```


* 단항으로 쓸 때
```cpp
printf("%d\n", *pA);  // pA가 가르키는 메모리를 참고하겠다. (pA 역참조)

```

review
```cpp

int* p; // 포인터 변수는 p이지, *p가 아니다!

p = &c; // O

*p = &c; // X

```

### 3. 포인터 practice
```cpp

int *p;
int i = 3, j;

p = &i;
j = *p;
j++;

printf("1. %d\n", *p); // 3
printf("2. %p\n", p); // i의 주소
printf("3. %d\n\n", *j); // 4

p = &j;

printf("4. %d\n", *p); // 4
printf("5. %p\n", p); // j주소

*p = *p + 3;

printf("6. %d\n", i); // 3
printf("7. %d\n", j); // 7
printf("8. %d\n", *p); // 7
```
