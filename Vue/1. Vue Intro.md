

### 1.ref
```jsx

<script setup>
	import {ref} from 'vue'
	
	const product = ref("socks");
</script>

<template>
	<h1>{{ product }}</h1>
</template>


```

### 2. v-bind
Dynamically bind an attribute to an expression

v-bind:src
:src 로 생략가능

:href
:alt
:

```jsx

<script setup>

import { ref } from "vue";
import socksGreenImage from "./assets/images/socks_green.jpeg";

const product = ref("Socks");
const image = ref(socksGreenImage);

</script>

  

<template>
	<img v-bind:src="image" />
	<img :src="image" />
</template>

```
