### 1. UML
- Unified Modeling Language
- Graphic 표기법
- 여러 다이어그램 정의
클래스 D, 패키지 D, 객체 D, 시퀀스 D....
- 소프트웨어 시스템 표현, 설계 사용
- 의사소통 도구로 활용하기 좋음

2. 클래스 다이어그램
- 객체의 타입을 표현
- 타입 간의 정적 관계 기술
- 설계할 때 자주 사용하는 다이어그램 중 하나
