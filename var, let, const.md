
var
- var 키워드로 선언한 변수는 중복 선언이 가능하다.
-  함수 레벨 스코프를 가진다.
- 변수 호이스팅


let
var 키워드로 선언한 변수와 달리 let 키워드로 선언한 변수는 변수 호이스팅이 발생하지 않는 것처럼 동작한다.

```js
console.log(foo); // ReferenceError: foo is not defined
let foo;

```
위 처럼, let 키워드로 선언한 변수를 변수 선언문 이전에 참조하면 참조 에러(ReferenceError)가 발생한다.

var 키워드로 선언한 변수는 런타임 이전에 자바스크립트 엔진에서 암묵적으로 "선언 단계", "초기화 단계"가 한번에 진행된다.
즉, 선언 단계에서 스코프(실행 컨텍스트의 렉시컬 환경)에 변수 식별자를 등록해 자바스크립트 엔진에 변수의 존재를 알린다. 그리고 즉시 초기화 단계에서 undefined로 변수를 촉
